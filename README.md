#  AI Contract Analyzer + Decision System

An automated AI-powered contract risk analysis system built with n8n.

##  Features
-  Auto PDF extraction from Google Drive
-  AI risk analysis using Google Gemini
-  Risk scoring (0-20) with priority levels
-  Slack alerts with Approve/Reject buttons
-  Gmail email notifications
-  Google Sheets audit trail

##  Tech Stack
- n8n (automation)
- Google Gemini AI
- Slack API
- Google Drive
- Google Sheets
- Gmail

##  Workflow
1. PDF uploaded to Google Drive
2. AI analyzes contract risks
3. Risk score + priority generated
4. Slack notification sent
5. Human approves or rejects
6. Google Sheets logged
7. Email notification sent
